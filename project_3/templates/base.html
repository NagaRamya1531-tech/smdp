Welcome to the Social Media Toxicity Analysis Dashboard

This dashboard is designed to analyze online discussions on immigration and refugees across social media platforms like Reddit and 4chan. Using advanced natural language processing techniques and real-time data collection, we measure toxicity, analyze sentiment, and identify patterns in public discourse. The goal is to provide insights into how sensitive topics like immigration are discussed online, empowering researchers and policymakers to address harmful narratives and foster constructive dialogue.
